# Política de Privacidad y Aviso Legal

Este documento detalla los términos de uso y el manejo de datos de BOE GPT en relación con el uso de la API de Datos Abiertos del BOE.

## 1. Naturaleza del Servicio
BOE GPT es un proyecto independiente, sin ánimo de lucro y de uso libre. No tiene vinculación oficial con la Agencia Estatal Boletín Oficial del Estado.

## 2. Uso de Datos y Privacidad
- **API del BOE**: El agente accede a la información a través de la API oficial de Datos Abiertos del BOE exclusivamente para lectura y consulta de información pública.
- **Datos Personales**: BOE GPT no recopila, almacena ni procesa datos personales de los usuarios. Las interacciones son efímeras dentro del contexto de la sesión de chat proporcionada por la plataforma GPT.
- **Seguridad**: No se realizan escrituras ni modificaciones en bases de datos externas a través de este agente.

## 3. Descargo de Responsabilidad (Disclaimer)
- **Finalidad Informativa**: La información proporcionada por BOE GPT tiene carácter meramente informativo y divulgativo. No tiene validez legal.
- **Exactitud**: Aunque se busca la máxima precisión, el usuario debe verificar siempre la información en la fuente original ([www.boe.es](https://www.boe.es)). El autor de este GPT no se hace responsable de las decisiones tomadas basadas en las respuestas del agente.
- **Sin Garantía**: El servicio se ofrece "tal cual", sujeto a la disponibilidad de la API del BOE y de la plataforma GPT.

## 4. Compromiso de Gratuidad
El autor se compromete a mantener este acceso de forma abierta y gratuita, cumpliendo con el espíritu de reutilización de la información del sector público (Ley 37/2007).
